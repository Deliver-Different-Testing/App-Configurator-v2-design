## Summary
This package proposes the new **System → Rating** module for Configurator and brings together:

- On-Demand
- Air Freight
- Scheduled / Routed
- Charges & Accessorials
- Break Pricing
- Units & Measures
- Service Rating Setup

The prototype uses the existing **centre list + right-hand drawer** pattern and adds a **progress-step setup flow** for linked rate components.
